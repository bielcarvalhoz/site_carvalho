## ✦ Sobre

Site estático de página única, escrito em **HTML, CSS e JavaScript puro** — sem frameworks, sem build, sem dependências. É só abrir no navegador.

## ✦ Destaques

| | |
|---|---|
| 🌐 **Rede de partículas** | Canvas animado que reage ao mouse e muda de cor conforme o scroll |
| 🎲 **Carrossel 3D** | Prisma de 4 faces (código, terminal, foto, stack) girado pelo scroll |
| 🌗 **Bilíngue** | Alternância instantânea entre Português e Inglês |
| ✨ **Reveal no scroll** | Animações suaves via `IntersectionObserver` |
| 🖱️ **Tilt 3D** | Cards de stack e projetos inclinam seguindo o cursor |
| 📱 **Responsivo** | Layout adaptado para mobile, tablet e desktop |
| ♿ **Acessível** | Respeita `prefers-reduced-motion` |

## ✦ Seções

- **Stage** — apresentação imersiva com carrossel 3D
- **Stack** — tecnologias do front ao banco de dados
- **Projetos** — trabalhos selecionados
- **Jornada** — trajetória profissional e acadêmica
- **Contato** — links e redes

## ✦ Como rodar

Não precisa de servidor nem instalação. Basta abrir o arquivo:

```bash
# abra index.html no navegador, ou sirva localmente:
npx serve .
```
